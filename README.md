# TheophilusX

TypeScript-based Discord bot framework with modular command and event handling.

## Documentation

- [Getting Started](docs/01-Getting-Started.md)
- [Adding New Commands](docs/02-Add-New-Commands.md)
- [Adding New Events](docs/03-Add-New-Events.md)

## Installation

```bash
npm install
```

## Building

```bash
npm run build
```

## Running

```bash
npm run start
```

## Configuration

Create a `.env` file in the root directory:

```env
DISCORD_TOKEN="YOUR_TOKEN_HERE"
GUILD_ID="YOUR_TEST_SERVER_HERE"
MONGODB_URI="YOUR_MONGODB_URI_HERE"
```
